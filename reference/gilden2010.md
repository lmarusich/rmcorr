# Repeated measurements of reaction time and accuracy

A dataset containing four repeated measurements of reaction time (RT)
and accuracy from eleven subjects in a visual search experiment. Each
measurement is the mean RT and accuracy from a block of 288 search
trials. blocks of visual search, for eleven subjects.

## Usage

``` r
gilden2010
```

## Format

A data frame with 44 rows and 4 variables

|        |         |                    |
|--------|---------|--------------------|
| `[,1]` | `sub`   | Subject ID         |
| `[,2]` | `block` | Block ID           |
| `[,3]` | `rt`    | Mean reaction time |
| `[,4]` | `acc`   | Mean accuracy      |

## Source

Gilden, D.L., Thornton, T.L., & Marusich, L.R. (2010). The serial
process in visual search. *Journal of Experimental Psychology: Human
Perception and Performance, 36*, 533-542,
[doi:10.1037/a0016464](https://doi.org/10.1037/a0016464)
